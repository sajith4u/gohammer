
# GoHammer

This project is to Run Simple API Load Tests Using Terminal

Following go packages are available

    1. fasthttp 
    2. tview 
    3. cobra



## Tech Stack


**Tool:** Go


## Run Locally

Clone the project

```bash
  git clone https://github.com/sajith4u/gohammer.git
```

Go to the project directory

```bash
  go build -o gohammer .
```

Start the Tool

```bash
  ./gohammer run
```
